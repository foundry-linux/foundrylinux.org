№ 02 · Install

Add the forge
to your sources.

Already running Ubuntu 26.04 LTS or any apt-compatible derivative? Add our signed repository and pull the Foundry kit piece by piece.

shell · root
① Trust the keyringcurl -fsSL https://apt.foundrylinux.org/keyring.gpg \ | sudo tee /etc/apt/keyrings/foundry.gpg > /dev/null② Wire up the repositoryecho "deb [signed-by=/etc/apt/keyrings/foundry.gpg] \ https://apt.foundrylinux.org stable main" \ | sudo tee /etc/apt/sources.list.d/foundry.list③ Update and installsudo apt update && sudo apt install foundry-base# or pick à la carte:sudo apt install world-foundry blender-foundry mame 65ax libvgm

Signed by the smith.

Every package in the Foundry archive is signed with our long-term maintainer key. Verify the fingerprint before adding the keyring to a production machine.
